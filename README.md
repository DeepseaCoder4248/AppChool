# 앱출-앱으로 시작하는 출석체크

<img width="100%" src="https://user-images.githubusercontent.com/98150988/218635545-b8c9515e-88bf-4852-aa6d-2fa2af6c4ecd.png"/>

# 설명
매일 앱 또는 웹을 실행해야하는데 귀찮으신 분
원하는 시간대에 실행시키고 싶으신 분
체계적으로 사용하고 싶으신 분
앱출이 그 고민을 해결해드립니다! 알람은 기본! 실시간 알람 기능까지!

■기능
1.내가 사용하는 앱이나 웹 사이트 링크를 등록하여 원하는 시간대와 요일을 설정하여 규칙적이고, 반복적인 기능을 자동화 시키는데에 도움이 됩니다.

2.알람이 울리면, 터치만 하면 해당 앱 또는 웹 사이트가 실행됩니다.

3.달력을 확인하여 얼만큼 출석이 잘 진행되고 있는지 확인이 가능합니다.

4.서버와의 연동으로 계정 연동이 가능하여, 다른 기기에서도 사용이 가능합니다.

■사용 예시
1. 쇼핑몰, 게임 출석등 정기적인 출석체크가 필요할 때

2. 운동하는 사람이나 핸드폰 켜기 귀찮으실 때

3. 정기적으로 제 시간에 맞추어서 실행해야 하는 것을 자동화 하고 싶을 때
